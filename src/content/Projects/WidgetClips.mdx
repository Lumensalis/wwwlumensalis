# Widget Clips


 Widget Clips is a system for mounting various small connected devices for prototyping.

It is based around a collection of 3D pritanble Clips and Mounts which are connected by a standardized Base Plate design.  
For more details, check out the [overview](https://gitlab.com/lumensalis/widgetclips/-/blob/main/docs/Overview.md)
and the [repo on gitlab](https://gitlab.com/lumensalis/widgetclips) : https://gitlab.com/lumensalis/widgetclips
